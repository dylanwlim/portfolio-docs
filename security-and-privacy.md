# Security And Privacy

Dylan W. Lim Portfolio Site should be used with the same care you would use for any live website or app.

## Good Habits

- Use listed contact paths for normal outreach.
- Do not send passwords, payment numbers, or unrelated personal records through portfolio contact channels.
- For a broken link or page issue, include the page URL and the device/browser used.

## Reporting A Concern

For corrections or questions, email [dylan@wlim.work](mailto:dylan@wlim.work). If the concern involves account access, payment, health, portfolio, transcript, or client information, email first so it can be handled carefully.
